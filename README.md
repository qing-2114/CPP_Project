## 环境要求

- 编译器：g++ (MinGW) 或兼容的 C++ 编译器
- IDE：VS Code
- 操作系统：Windows

## 如何运行

### 方法1：使用 VS Code 终端

```powershell
cd "e:\C++_Project\C++基础\类与对象"
g++ -fexec-charset=GBK -finput-charset=UTF-8 "23.加号运算符重载.cpp" -o "23.加号运算符重载.exe"
.\23.加号运算符重载.exe
```

### 方法2：使用 VS Code 任务

1. 按 `Ctrl + Shift + P` 打开命令面板
2. 输入 `Tasks: Run Task`
3. 选择 `C/C++: g++ run active file`
